# Profit Analysis Using Multiple Linear Regression

This project analyzes how different company investments affect profit using a Multiple Linear Regression model.

## Project Objective
The goal of this project is to identify which spending categories contribute the most to company profit.

## Dataset
The dataset contains information about 50 startup companies with the following variables:

- R&D Spend
- Administration Spend
- Marketing Spend
- Profit
- State

## Tools Used
- Excel
- Python / Regression Analysis
- Power BI
- Statistical Modeling

## Analysis Performed
- Data cleaning and validation
- Correlation analysis
- Multiple Linear Regression modeling
- Profit prediction scenarios
- Power BI dashboard visualization

## Key Insights
- R&D spending has the strongest impact on profit.
- Marketing spend shows moderate influence.
- Administration spending has minimal impact.
